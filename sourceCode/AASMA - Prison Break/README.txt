To run the program you must first install:
 - Unity Hub
 - Unity version 2022.3.26f1 LTS

Then you can open the project. 
Once in the editor you can run the simulation by pressing the play button on the top center of the screen.

To change the number of prisoners and guards:
 - In the project explorer, go under Scenes -> Test Scenes, and double click the one you want to open.

To change the behavior of the agents:
 - Select the prisoner object and in the "Prisoner (Script)" section in the inspect change the ticks for "Can Bribe" and "Can Incite", to determine what the agent can do.

(The prisoners are located under "Agents" in the scene explorer)
